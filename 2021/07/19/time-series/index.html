Univariate Stationary Processes

Autoregressive Processes

A first order autoregressive process, an $AR(1)$ process, can be written as an inhomogeneous stochastic first order difference equation:

$$x_t = \delta + \alpha x_{t-1} + u_t$$

The inhomogeneous part $\delta + u_t$ consists of a constant term $\delta$ and a pure random process $u_t$. We can express the relation by using the lag operator defined as $\text{L} x_t = x_{t-1}$:

$$(1 - \alpha \text{L})x_t = \delta + u_t$$

If we solve for $x_t$ we get he Wold representation with $\phi_j = \alpha^j$ and $| \alpha | < 1$:

$$x_t = \frac{\delta}{1 - \alpha} + \sum_{j=0}^{\infty} \phi_j u_{t-j}$$
